Build the Accounting module for FinanceOS using the attached mockup as the primary design reference.

This is NOT QuickBooks.
This is the FinanceOS Accounting Workspace.

Goal:
Create a modern accounting workspace focused on daily operational work, while keeping all reporting inside the Reporting module.

Architecture
------------
This is a standalone FinanceOS module.

Route:
/accounting

All pages should use the same Accounting layout.

Do NOT include Reporting, Budget, Forecast or Commission navigation inside this module.

The sidebar is Accounting-only.

Design Requirements
-------------------

The Accounting module must visually match the existing FinanceOS Portfolio dashboard.

Same spacing

Same typography

Same border radius

Same shadows

Same colors

Same component library

Same header

Same sidebar behavior

Do NOT use QuickBooks colors or branding.

This should feel like a premium internal finance operating system.
